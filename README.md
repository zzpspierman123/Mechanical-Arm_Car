## ⭐️About Project

**基于工训大赛中智能物料搬运小车所设计的机械臂小车** 
<div>
 
***机械结构：*** 采用Solidwork进行3D建模。
<div>

***视觉识别：*** 采用Openmv单片机。
<div>

***远程控制：*** 采用ESP32和上位机。
<div>

***主控：*** 采用STM32F407单片机。
<div>

***6自由度机械臂：*** 采用Matlab进行机械臂仿真。
<div>
